Logos

The "nStack" logo is used throughout the backends and inherits the $primary or $secondary colors.
You can combine any of the classes listed below to modify the look of the logo, eg. <div class="logo logo-secondary logo-flat">.

Class Description
.logo Default logo size and primary gradient color
.logo-small Small logo size
.logo-large Large logo size
.logo-xlarge Extra large logo size
.logo-secondary Secondary logo color
.logo-flat Non-gradient color (primary or secondary)

Note: Because we use inline SVG's and Gradients you can only have one logo variation per page.
