    Welcome to the Nodes UI Framework

    The documentation is still a work in progress, and currently we are focusing all our efforts on implementing the actual code instad of documenting it. Please bear with us.

    Contributing

    We have a few rules you absolutely need to adhere to if you want to contribute to the Nodes UI Framework:

    • ANY styleguide specific component must be styled by using a prefixed class ie.: class="docs--my-styleguide-button"
    • Make sure to follow existing pages markup structure. Especially xray-html is a little tricky.
    • If your example requires some specific styling, write that in a <style> directly in the document.
